# farm-management-system
A custom WordPress plugin built to manage livestock, crops, vaccinations, tasks, and reporting for modern farm operations.

# Features

- Animal management with health tracking
- Vaccination scheduling with automated email reminders
- Crop and inventory management
- Reporting & analytics dashboard
- Internal support module

# Built With

- PHP
- WordPress
- MySQL
- Custom plugin architecture (OOP structure)
